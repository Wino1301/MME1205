COURSE DESCRIPTION#

MME 1205 (Graphics) is a required 2-credit course (1 hours of lecture and 3 lab hours).

Three broad topics will be covered in this class:

  1. An introduction to the Python programming language

  2. Applications of Python programming to problems that are unique to Metallurgical, Materials and Biomedical Engineering (MMBME) including numerical analysis of experimental data, quantitative image analysis and basic statistical analysis of material response

  3. An introduction to digital manufacturing and engineering drawing software using Solidworks The goal of this course is to establish a foundation for numerical analysis, digital manufacturing and computational methods that can be carried forward throughout the rest of the MMBME curriculum.
